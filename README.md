# xisan.tech

Personal portfolio website and digital sandbox.

Live at: [xisan.tech](https://xisan.tech)

## Overview
A static frontend portfolio built without any heavy frameworks. It uses vanilla HTML/CSS/JS and a slightly customized Neofetch implementation for the hero section to keep things lightweight and terminal-themed.

Forms are handled via Formspree, and the site is hosted on GitHub Pages with Cloudflare managing DNS.

## Stack
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JS 
- FontAwesome

## Setup
No build process. Just clone and open `index.html`.

```bash
git clone https://github.com/Xeesan/Xeesan.github.io.git
cd Xeesan.github.io
```
